# SCA Catalog Placeholder

Esta es una carpeta para el catálogo PDF de SCA.

## Uso

Coloca tu archivo PDF del catálogo aquí con el nombre: `SCA_Catalog.pdf`

El archivo será accesible desde la página de conexión QR cuando los visitantes del evento presionen "View Catalog".

## Ruta de acceso

La ruta configurada es: `/documents/SCA_Catalog.pdf`

Si deseas cambiar el nombre del archivo, edita:
- `src/data/connectData.ts` 
- Actualiza la constante `catalogPDF`
